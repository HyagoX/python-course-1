# Aula 2: Operadores em Python üêç

## üìã Conte√∫do da Aula

Esta aula abordou os diferentes tipos de operadores dispon√≠veis em Python, fundamentais para manipular dados e criar l√≥gica em nossos programas.

## üìö Operadores Estudados

### 1. Operadores Aritm√©ticos (`operadores_aritmeticos.py`)

Os operadores b√°sicos para realizar c√°lculos matem√°ticos:

- **Adi√ß√£o (`+`)**: Soma dois valores
- **Subtra√ß√£o (`-`)**: Subtrai o segundo valor do primeiro
- **Multiplica√ß√£o (`*`)**: Multiplica dois valores
- **Divis√£o (`/`)**: Divide e retorna um **FLOAT**
- **Divis√£o Inteira (`//`)**: Divide e retorna apenas a parte inteira (**INT**)
- **M√≥dulo (`%`)**: Retorna o resto da divis√£o
- **Exponencia√ß√£o (`**`)**: Eleva um n√∫mero a uma pot√™ncia

```python

print(10 / 2)    # 5.0 (float)
print(10 // 2)   # 5 (int)
print(10 % 3)    # 1 (resto da divis√£o)

```

### 2. Operadores de Associa√ß√£o (`operadores_associacao.py`)

Verificam se um objeto existe dentro de uma sequ√™ncia:

- **`in`**: Verifica se existe
- **`not in`**: Verifica se N√ÉO existe

‚ö†Ô∏è **Importante**: S√£o **case sensitive** (diferencia mai√∫sculas de min√∫sculas)

```python

```

### 3. Operadores de Atribui√ß√£o (`operadores_atribuicao.py`)

Permitem atribuir e modificar valores de vari√°veis:

- **`=`**: Atribui√ß√£o simples
- **`+=`**: Adi√ß√£o e atribui√ß√£o
- **`-=`**: Subtra√ß√£o e atribui√ß√£o
- **`*=`**: Multiplica√ß√£o e atribui√ß√£o
- **`/=`**: Divis√£o e atribui√ß√£o
- **`//=`**: Divis√£o inteira e atribui√ß√£o
- **`%=`**: M√≥dulo e atribui√ß√£o
- **`**=`**: Exponencia√ß√£o e atribui√ß√£o


### 4. Operadores de Compara√ß√£o (`operadores_comparacao.py`)

Comparam valores e retornam `True` ou `False`:

- **`==`**: Igualdade
- **`!=`**: Diferen√ßa
- **`>`**: Maior que
- **`>=`**: Maior ou igual
- **`<`**: Menor que
- **`<=`**: Menor ou igual


Verificam se duas vari√°veis ocupam o mesmo espa√ßo na mem√≥ria:

- **`is not`**: Verifica se N√ÉO ocupam o mesmo espa√ßo

### 6. Operadores L√≥gicos (`operadores_logicos.py`)

Combinam express√µes l√≥gicas:

- **`and`**: Retorna `True` apenas se ambas as condi√ß√µes forem verdadeiras
- **`or`**: Retorna `True` se pelo menos uma condi√ß√£o for verdadeira
- **`not`**: Inverte o valor l√≥gico


### 7. Preced√™ncia de Operadores (`precedencia.py`)

Python segue as regras matem√°ticas de preced√™ncia:

1. **Par√™nteses** `()`
2. **Exponencia√ß√£o** `**`
3. **Multiplica√ß√£o e Divis√£o** `*`, `/`, `//`, `%` (da esquerda para direita)
4. **Adi√ß√£o e Subtra√ß√£o** `+`, `-` (da esquerda para direita)


## üéØ Conceitos Principais Aprendidos

- **Diferen√ßa entre `/` e `//`**: Divis√£o comum vs divis√£o inteira
- **Case Sensitivity**: Python diferencia mai√∫sculas de min√∫sculas
- **Preced√™ncia**: A ordem correta de execu√ß√£o das opera√ß√µes
- **Operadores de Identidade vs Igualdade**: `is` verifica mem√≥ria, `==` verifica valor
- **Combina√ß√£o de Operadores L√≥gicos**: Uso de par√™nteses para clareza
